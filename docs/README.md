# 📱 Пробное приложение: Техническое задание

## 🧩 Структура приложения

Приложение содержит **три экрана**:

1. **Обучающие уроки** — показ материалов (текст, видео, изображения).
2. **График и сделки** — пользователь открывает сделки на 1 или 5 минут, выбирает направление (вверх/вниз), как в бинарных опционах.
3. **Брокеры** — список брокеров с переходами по ссылкам (WebView или внешний браузер).

## 2. 📈 Экран с графиком и бинарными сделками

- График цены (TradingView, investing.com или собственный через API).
- Пользователь:
  - Выбирает срок сделки: **1 минута** или **5 минут**
  - Выбирает направление: **⬆ Вверх** / **⬇ Вниз**
- Сделка фиксируется → запускается таймер → по истечении времени — закрытие сделки.
- Отображение результата: победа / поражение.
- Награда начисляется виртуально (без использования реальных денег).
- График реализуется через **Flutter chart библиотеку** syncfusion_flutter_charts

![alt text](docs/assets/scr2_ref1.jpg)

Экран предоставляет пользователю возможность **анализировать график в реальном времени** и **открывать сделки "вверх/вниз"** на ограниченное время (1 или 5 минут), как в бинарных опционах.

---

### 🧠 Основная идея

- Пользователь выбирает актив (например, Bitcoin)
- Анализирует поведение графика
- Выбирает срок сделки (например, **1 минута**)
- Нажимает кнопку **"Buy ↑"** или **"Sell ↓"**
- Через 1 или 5 минут сделка закрывается и начисляется награда (или нет)

### ✅ Требуемая механика для нашего приложения

1. Пользователь выбирает **время сделки**: `1 минута` или `5 минут`
2. Нажимает кнопку **вверх (Buy ↑)** или **вниз (Sell ↓)**
3. Приложение:
   - фиксирует текущую цену и направление
   - запускает таймер (60 или 300 сек)
   - по завершении сравнивает цену
   - определяет, выиграл ли пользователь
   - начисляет виртуальную награду
4. Сделки могут отображаться в виде списка (активные / завершённые)

### 📦 Реализация графика

- **Flutter chart библиотека** (например, `syncfusion_flutter_charts`) + API (CoinGecko, Binance, Investing.com) — гибко, но требует больше времени

### 🧮 Логика сделки (псевдокод)

```dart
void openDeal(Direction direction, Duration duration) {
  final entryPrice = getCurrentPrice();
  final endTime = DateTime.now().add(duration);
  startTimer(endTime, () {
    final exitPrice = getCurrentPrice();
    final won = (direction == Direction.up && exitPrice > entryPrice) ||
                (direction == Direction.down && exitPrice < entryPrice);
    showResult(won);
  });
}
